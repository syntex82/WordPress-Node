## Installation

### Method 1: Upload via Admin Panel

1. Download the theme ZIP file
2. Go to your WordPress Node admin panel
3. Navigate to Settings â†’ Themes
4. Click "Upload Theme"
5. Select the ZIP file and upload
6. Click "Activate" on the Modern Blog theme

### Method 2: Manual Installation

1. Download the theme ZIP file
2. Extract it to your `themes/` directory
3. Go to Settings â†’ Themes in the admin panel
4. Click "Scan Themes"
5. Click "Activate" on the Modern Blog theme

## Customization

### Colors

Edit the CSS variables in `assets/css/style.css`:

```css
:root {
  --color-primary: #2563eb;
  --color-secondary: #64748b;
  --color-accent: #f59e0b;
  --color-background: #ffffff;
  --color-text: #1e293b;
}
```

### Fonts

The theme uses Inter font by default. You can change it in `theme.json` or the CSS variables.
